# Cooking

*烹饪*操作能够将**多个**食材处理成**菜肴**,但也可能将**菜肴**烤糊变成黑暗料理.

The *cooking* operation can process **multiple** ingredients into **cusines**, but it may also burn **cusines** into bad cusines.
